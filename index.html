e-Journal

  • VB6

The application I used to monitor the daily activities in my area of responsibilities. The purpose of this program are as follows:

  • To have a database where I can save all the reports so I can easily search it when it's needed.
  • To identify which town(s) have the highest no. of theft incident in our area so we can deploy SRF (Security Response Force) to apprehend the suspect(s) / to secure the area.
  • To view history of each facility. So incase there's an incident happened to a particular facility, we have a basis / reference for our investigation report.
  • To monitor the performance of every SRF / Cluster Roving Guards.
